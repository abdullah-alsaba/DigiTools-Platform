# DigiTools Platform вЂ“ Digital Tools Buying Website

DigiTools Platform is a modern **Digital Tools Marketplace Website** built using **React (Vite)**, **Tailwind CSS**, and **DaisyUI**.
Users can explore digital tools, add them to the cart, remove products, and complete checkout easily.

This project was developed for **Assignment 6**.

## How to Run the Project

### Clone the repository

```id="qf5hh1"
git clone https://github.com/abdullah-alsaba/DigiTools-Platform.git
```

### Go to project directory

```id="s7y1w1"
cd DigiTools-Platform
```

### Install dependencies

```id="qg0q6n"
npm install
```

### Run the development server

```id="o0st8g"
npm run dev
```

Project will run on:

```id="4v4p0b"
http://localhost:5173
```
